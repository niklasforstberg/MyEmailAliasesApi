- Named **Peach** (previously Trysil)
- Palette: soft-apricot #f9dbbd → cotton-candy #ffa5ab → blush-rose #da627d → berry-crush #a53860 → night-bordeaux #450920
- Variables are purpose-driven (not appearance-based): `--surface-*`, `--action`, `--border-*`
- Key semantic vars: `--surface-inverse/#450920` (dark panels), `--action/#da627d` (buttons/links), `--surface-page/#fef7f1` (bg)
- Focus ring rgba: `rgba(218, 98, 125, 0.15)` (blush-rose based)
- Brand book: `docs/brand.html` (standalone, open directly in browser)
- Key files: `clientapp/src/index.css` (design system), `clientapp/src/App.css`, `clientapp/src/components/AliasesList.css`, `clientapp/src/components/Login.css`
