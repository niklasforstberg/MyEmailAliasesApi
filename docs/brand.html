Color Palette

Five raw palette colors — the source of truth. All semantic variables reference these. Never use raw hex values directly in component CSS; always go through a semantic variable.

Surfaces & Borders

Purpose-driven variables for backgrounds and borders. These are what you reference in CSS — not the raw palette colors.

Variable Swatch Value Used for
--surface-page #fef7f1 Main page background
--surface-subtle #fceade Disabled inputs, empty-state fills, scrollbar track
--white #ffffff Cards, modals, text on dark surfaces
--surface-inverse #450920 (night-bordeaux) Header, alias cards — dark base
--surface-inverse-mid #6b1535 Header, alias cards — gradient highlight pair
--border-subtle #f0d5c0 Default input and card outlines
--border-default #dbbda8 Hover borders, dividers, scrollbar thumb

Typography

Two fonts, two distinct roles. Cormorant Garamond carries identity and hierarchy; Nunito handles all UI and readable text.
